# 💰 Expense Tracker

A responsive and user-friendly Expense Tracker built using **HTML, CSS, and JavaScript**. This app helps you manage daily income and expenses, track your balance, and stores all data using Local Storage so nothing is lost on refresh.

## 🚀 Features

- Add Income and Expense Transactions
- Real-Time Balance, Income & Expense Summary
- Recent & Full Transaction History
- Spending Breakdown by Category
- Analytics with Cash Flow Chart
- Monthly Budget Tracker with Progress Bar
- Dark Mode Support
- Local Storage (Data persists after page refresh)
- Fully Responsive (Mobile Friendly)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Browser Local Storage
